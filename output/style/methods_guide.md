# Methods Writing Guide for Nature Communications

## Tense Usage
- Procedures: Past tense (>95%)
- Definitions: Present tense (<5%)

## Key Vocabulary (Top 40)
### Nouns: soil, warming, figure, temperature, soils, emissions, effect, fluxes, mineralization, nitrification, denitrification, cycling, effects, availability, plants
### Verbs: increase, warm, wet, dry, download, see, govern, induce, affect, show, elevate, indicate
### Adjectives: microbial, elevated, higher, organic, high, applicable, global, significant
### Adverbs: also, significantly, however, thus, respectively

## Sentence Structure
- Average length: 21.46 words
- Prefer longer, detailed sentences for procedures
- Sentence types: Simple 20%, Compound 40%, Complex 40%
- Length distribution: Short 15%, Medium 60%, Long 25%

Coordinating: and, or (technical coordination)

## Structure
1. Study design overview
2. Participants/materials description
3. Experimental procedures
4. Statistical analysis
5. Ethical statements

## Style Notes
- Passive voice ratio: 0% (high for objectivity)
- Detailed enough for reproducibility
- Include sample size calculations if applicable
- Use precise technical vocabulary
- Avoid vague terms: may, could, suggest, indicated, tend
